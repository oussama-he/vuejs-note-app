# VueJS Note App
A simple note app that uses VueJS and localStorage of browser to create and edit notes in markdown format.
![](./preview.gif)
## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
